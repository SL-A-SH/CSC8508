7. Core API Reference | PS5 Specific

APIs for PS5 platform, see fmod_ps5.h

  • FMOD_PS5_AcmConfigure Control whether to offload to hardware and to set the internal float type for the Acm convolution reverb.
  • FMOD_PS5_AjmConfigure Allow memory expansion for the opus codec.
  • FMOD_PS5_AT9Configure Control whether to allocate AT9 codec hardware resources.

FMOD_PS5_AcmConfigure

Control whether to offload to hardware and to set the internal float type for the Acm convolution reverb.

C
C++
C#

FMOD_RESULT FMOD_PS5_AcmConfigure(
  FMOD_BOOL offloadConvolution,
  FMOD_BOOL halfPrecisionFloat
);

RESULT PS5.AcmConfigure(
  bool offloadConvolution,
  bool halfPrecisionFloat
);

Not supported for JavaScript.

offloadConvolution

Enable offloading of convolution reverb processing.

  • Units: Boolean
  • Default: True
halfPrecisionFloat

Use half float precision on the Acm hardware.

  • Units: Boolean
  • Default: False

Call before System::init or selection will not apply.

If offloadConvolution is set to true, then during the first System::init it will configure the Acm hardware (which happens by default), after which it cannot be changed until the last System::close.
If set to false the Acm hardware will be disabled for convolution reverb and the software version will be used instead.

If halfPrecisionFloat is set to true then the Acm hardware will use half float precision internally and the quality of playback will be slightly compromised in quality. Good for saving memory resources.
If set to false it will use full float precision internally.

FMOD_PS5_AjmConfigure

Allow memory expansion for the opus codec.

C
C++
C#

FMOD_RESULT FMOD_PS5_AjmConfigure(
  int expandedMemorySize
);

RESULT PS5.AjmConfigure(
  int expandedMemorySize
);

Not supported for JavaScript.

expandedMemorySize

Expands memory in MiB sizes upto 32 and must be powers of 2.

  • Range: [0, 32]
  • Default: 0

Call before System::init or selection will not apply.

If expandedMemorySize is set to 0 (which happens by default), then this will allow 165 mono opus streams to be active at any one time. If you need more streams, the memory needs to be expanded. Once this value has been set before System::init, it cannot be changed until System::close has been called. The FMOD Engine will log an error when memory needs to be expanded.

FMOD_PS5_AT9Configure

Control whether to allocate AT9 codec hardware resources.

C
C++
C#

FMOD_RESULT FMOD_PS5_AT9Configure(
  FMOD_BOOL enable
);

RESULT PS5.AT9Configure(
  bool enable
);

Not supported for JavaScript.

enable

Enable AT9 hardware resources to be allocated.

  • Units: Boolean
  • Default: True

Call before System::init or selection will not apply.

If enable is set to true, then during the first System::init it will allocate hardware resources for the AT9 codec (which happens by default), after which it cannot be changed until the last System::close.
If set to false the AT9 codec will not allocate hardware resources until an AT9 format file is loaded in.
